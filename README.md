## Additional setup

### Font

- Fire code nerd font mono
  - `brew tap homebrew/cask-fonts`

### FZF

- Install FZF
  - `brew install fzf`

### ripGrep

- [ripGrep home page](https://github.com/BurntSushi/ripgrep)

  - `brew install ripgrep`

### FD

- `brew install fd`
